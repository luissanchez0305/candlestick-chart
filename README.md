# Candlestick chart

Random calculation of values to display a daily candlestick stock chart

## Framework and Libraries

    - "react": "^18.2.0"
    - "@canvasjs/charts": "^3.7.36"
    - "@canvasjs/react-charts": "^1.0.0"

## Requirements

    - node v18.17.1

## Instructions

    - `npm i`
    - `npm start`
